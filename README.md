#test1.py
Check for maximum jitter and latency


##Command Format
```
usage: check-path.py [-l latency] [-j jitter]

optional arguments:
  -l latency
        Max latency defaults to 50 if not specified
  -j jitter
        Max jitter defaults to 50 if not specified
```
